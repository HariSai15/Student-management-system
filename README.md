# Student Management System (Java)

## Description
A console-based Java application to manage student records.
The system allows users to add, view, update, and delete student details.
Student IDs are unique to maintain data integrity.

## Features
- Add student
- View all students
- Update student details
- Delete student record
- Duplicate ID validation
- Menu-driven console interface

## Technologies Used
- Java
- OOPS concepts (Encapsulation, Inheritance, Polymorphism)
- Collections (ArrayList)
- Scanner class

## How to Run
1. Compile the program:
   javac StudentManagementSystem.java
2. Run the program:
   java StudentManagementSystem
